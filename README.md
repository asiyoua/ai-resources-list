## 💡 核心概念

### 什么是 Agent Skills？

Agent Skills 把领域特定知识、工作流程、最佳实践打包成可重用的"技能包"，让通用 AI Agent 转变为专精于特定任务的 Agent。

### 核心优势

| 优势 | 说明 |
|------|------|
| 专精化 | 将 AI Agent 转变为特定领域专家 |
| 复用性 | 一次创建，跨会话自动使用 |
| 可组合 | 支持技能组合，构建复杂工作流 |
| 高效性 | 渐进披露机制，仅加载相关内容 |

### 渐进披露机制

- **级别1：元数据**（始终加载）- name + description，几乎无 token 消耗
- **级别2：主要指令**（触发时加载）- SKILL.md 正文，通常 <5k tokens
- **级别3：资源与代码**（按需加载）- 额外文件、脚本，无上限限制

### Agent vs Skill vs Tool

| 概念 | 作用 | 类比 |
|------|------|------|
| Agent | 主动执行 | 大脑 |
| Skill | 知道怎么做 | 培训手册 |
| Tool | 具体工具 | 手的工具 |
